DATA DIRECTORY — README

This folder contains the input dataset used for the supply chain optimization model.

📁 File: Project_Data.xlsx

This Excel file contains the manually reconstructed dataset based on the information published in the reference article:

- Gomes & Rosa (2022) — A Bi-level Optimization Approach for the Location of Distribution Centers Supporting Offshore Oil Platforms

⚠️ Important Notes:
- The original dataset was not provided by the article's authors.
- All data was extracted and restructured manually from published tables for academic and demonstrative purposes only.
- No original or proprietary material from the article is distributed in this repository.

📌 This data is intended solely for educational and non-commercial use.
